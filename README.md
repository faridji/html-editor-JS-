# HTML Editor (JS)
This is an html editor build using html content editable property. 

## Options
1)  Table (add row above, add row below, add column left, add column right, remove row, remove column)
2)  List
3)  Image
4)  Hyperlink
5)  Background color
6)  Strikethrough
7)  bold
8)  Italic 
9)  Underline
10) Select All
